# mensajes-aleatorios-en-str
Es un paquete que retorna mensajes de tipo String de forma aleatoria.
## Install

```npm
npm install -g mensajes-aleatorios-en-str
```

# Usage

```bash
 mensajes-aleatorios-en-str
```
